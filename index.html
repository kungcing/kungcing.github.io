Sensitive Area Survey APP

An Android application designed to support sensitive environmental area survey work. In the S1 Conssession Oil field project area, covering 4 provinces in northern Thailand, simple operations by taking pictures and collecting information about their basic characteristics, such as building type, water, location, etc., are effective. reduce the number of surveyor

The app is developed by Android Studio. Details are as follows.

  • Main libraries
  • - CameraX, Android Jetpack Support Lib
  • - Picasso Lib
  • - FusedLocationProviderClient

  • App Features
  • - support for Android version 5.0 (lolipop) and up
  • - Take photos and save data in one touch
  • - Save raw data in *.csv file format for ease of use
  • - Auto save lattitude and lontitude coordinates system to *.csv file format
  • - Wireless data transfer via SAAD app

3D web-base Point Cloud

If you're looking for a 3D GIS development tool, this is a possibility. The 3D Web-Base Point Cloud is the perfect combination of point cloud data from field surveys, such as from drone data or 3D laser scanners, then generated on the server.
